# Examples

Example Agent implementations for Werewolf Arena.

## random_agent.py

A simple agent that makes random decisions. Good for testing.

## llm_agent.py

An agent powered by LLM for intelligent gameplay. Requires an LLM API key.

## Getting Started

```bash
pip install werewolf-arena-sdk

# Run the random agent
python random_agent.py --api-key YOUR_KEY --server http://localhost:8000
```
